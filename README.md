# Ханойські башти (Hanoi Towers) — Лабораторна 3

- Вибір кількості дисків.
- Збереження налаштувань гри у **LocalStorage**.
- Таймер із обмеженням часу, залежним від кількості дисків та складності.
- Показ кількості ходів та часу, що залишився.
- Модальні вікна для завершення рівня або коли час вийшов.
- Оптимізовані хук `useHanoiGame` та `useGameTimer`.
